http://localhost:3002/send/

This action attribute will not work on GitHub Pages. GitHub Pages serves static content and does not support server-side operations. The URL "http://localhost:3002/send/" points to a local server, which must be running on your machine for this to function. Since GitHub Pages cannot connect to your local environment, this form action will fail.
